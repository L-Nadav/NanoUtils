# NanoUtils
Utilities for Minecraft 1.12.2

## Requirements

JDK 8

## Build Instructions

1. Clone the repository
2. Windows - Run gradlew.bat clean build on command line
2. Unix - Run ./gradlew clean build on terminal
3. Done :) it's in build/libs/
